# SourceDeck 1.0.0

First public desktop release.

## Downloads

- **macOS (Apple Silicon / M-series):** `SourceDeck-1.0.0-arm64.dmg`
- **macOS (Intel x64):** `SourceDeck-1.0.0-x64.dmg`
- **Windows (x64):** `SourceDeck-Setup-1.0.0.exe`

## What's included

- Full SourceDeck Lead Command Center engine (ARCG-grade concept-translation + MedPilot vertical + SourceDeck GovCon vertical).
- Offline-capable operation; API keys stored via macOS Keychain / Windows Credential Manager through Electron `safeStorage`.
- Auto-updater wired to the `sourcedeck-releases` GitHub repo.

## Install

- **macOS:** mount the DMG, drag SourceDeck.app to /Applications. First launch: right-click → Open (unsigned build).
- **Windows:** run the NSIS installer. Choose install location when prompted.
