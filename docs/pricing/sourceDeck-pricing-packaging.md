# SourceDeck GovCon Pricing and Packaging

This is an internal packaging reference. It does not edit public
website pricing.

Required positioning for GovCon content and marketing prompts:

> SourceDeck helps small businesses organize GovCon pursuit workflows,
> prepare review-ready outreach/content, and manage capture activity
> with human approval at every decision point.

## Core — $1,497 one-time

Core is the self-install entry package.

- Self-install
- Basic GovCon workflows
- Basic templates
- Basic prompt library
- User-led setup

Core is for users who want the SourceDeck foundation and are willing to
configure their own profile, credentials, targeting, and workflow
habits.

## Growth — $3,497 one-time

Growth is Core plus guided setup and basic validation.

- Core package included
- Guided setup
- GovCon Operating Profile setup
- SAM/API onboarding support
- Basic workflow testing
- Walkthrough and handoff

### Difference between Core and Growth

Core gives the user the product foundation and templates. Growth adds
guided setup, operating-profile configuration, credential onboarding
support, basic workflow testing, and a walkthrough so the user starts
with less implementation friction.

## White-Glove — $5,997 one-time now

Raise White-Glove to $7,500 after traction.

White-Glove is the done-for-you implementation package.

- Done-for-you implementation
- Business profile setup
- Capability statement review
- NAICS/PSC/certification setup
- SAM/API/AI credential onboarding support
- Capture Suite setup
- Outreach Agent setup
- Prime Partner Finder setup
- Premium Content setup
- QA and handoff

### What exactly White-Glove includes

White-Glove covers the setup work that determines whether SourceDeck is
useful in the first week: business profile structure, capture workflow
configuration, targeting context, credential onboarding support,
content-prep setup, outreach draft setup, partner-finder setup, and a
QA handoff. It does not transfer decision authority to SourceDeck.
Operators still review and approve outreach, content, capture, pricing,
proposal, and pursuit decisions.

## Enterprise / Government — custom quote

Enterprise and Government pricing is custom because deployment,
governance, and security requirements vary.

- Private deployment
- Security review
- Tenant policy
- Custom workflows
- Governed AI provider rules

## Why monthly Operator pricing is justified

The one-time packages install the operating system. Operator packages
keep it useful as solicitations, agency targets, content needs, prompt
rules, and workflow habits change. Monthly value comes from continued
optimization, prompt maintenance, draft review/support, content planning,
workflow tuning, and campaign support, not from autonomous action.

## Why old pricing is underpriced

Older pricing treated SourceDeck like a static tool or template bundle.
The current GovCon surface includes capture workflow organization,
SAM/API onboarding support, outreach draft preparation, prime-partner
research support, Premium Content setup, credential-boundary hardening,
review gates, smoke checks, and QA/handoff paths. That implementation
and operating discipline is materially more valuable than a simple
search wrapper or prompt pack.

## Third-party API costs remain client-owned

Third-party API costs remain client-owned because each customer controls
their own usage, credentials, limits, providers, and vendor terms.
SourceDeck can help configure and route those credentials safely, but it
should not absorb variable customer-specific costs for SAM/API, AI,
creative, enrichment, hosting, or other external services.

## Human approval is part of the value proposition

Human approval protects the customer. GovCon work involves procurement
integrity, solicitation communication rules, claim support, pricing
judgment, proposal responsibility, and capture decisions that should not
be delegated to software. SourceDeck prepares review-ready workflows and
drafts so the operator can move faster while remaining the final
decision-maker.

## Overclaiming guardrails

- Do not claim SourceDeck wins contracts.
- Do not claim guaranteed awards, revenue, compliance, or outreach
  success.
- Do not claim auto-send, auto-post, or autonomous GovCon
  decision-making.
- Do not weaken human approval language.
