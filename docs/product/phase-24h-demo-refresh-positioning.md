## What SourceDeck is

- A **GovCon Capture OS** — a desktop workspace for federal pursuit planning.
- An **end-to-end capture workflow**: Capture Command Center → Operating Rhythm → Deadline & Q&A → Pre-RFP → Agency Targeting → Solicitation Workspace → Compliance Matrix → Vendor Quote Room → Pricing Worksheet → Past Performance Library → Capability Statement Studio → Prime Partner Finder → Stakeholder Graph → Submission Readiness Gate → Internal Review Export → Audit Log.
- A **human-in-the-loop** tool: it organizes, drafts, and reviews; the user decides and acts.
- **Local-first**: capture/workspace state is local to the device; the internal-review export is a local Markdown file.

## What SourceDeck is NOT

- Not an autopilot — it does **not** submit bids, quotes, or proposals.
- Not an email sender — it does **not** send outreach or contact officials.
- Not a portal integration — it does **not** upload to SAM.gov / PIEE / eBuy / GSA / agency portals.
- Not a compliance certification — it is **not** FedRAMP / SOC 2 / CMMC / HIPAA / HITRUST / ISO certified, and does not claim to be.
- Not a guarantee — it does **not** promise awards or revenue.
- Not a signed/notarized public release in current builds (unsigned dev builds; do not claim signed/notarized/Apple-notarized without exact release evidence).
- Not a replacement for legal, contracting, compliance, or proposal professionals.

## Buyer value bullets

- One workspace for the whole capture workflow — fewer dropped pursuits.
- Faster solicitation shred + a per-requirement compliance matrix.
- Teaming readiness (vendors, primes) with manual, human-controlled outreach status.
- Reusable past performance + tailored capability statements (local/offline import).
- FAR-aware stakeholder mapping with restricted-window awareness — engage compliantly.
- Advisory submission readiness + a local internal-review package — confidence before the user submits externally.
- A local audit log — the operator's own evidence trail.
