# SourceDeck Cloudflare Secure Download Worker

## Purpose

This Worker is the **paid customer delivery gate** for SourceDeck macOS release artifacts. It is designed to sit between Lemon Squeezy purchase events and private Cloudflare R2 release files. Customers should receive a short-lived download link after purchase, not a permanent public storage URL.

## Required Cloudflare Resources

| Resource | Purpose |
|---|---|
| Cloudflare Worker | Receives Lemon Squeezy webhooks and serves token-gated downloads. |
| Private R2 bucket | Stores signed/notarized `.dmg`, `.zip`, checksum, and release evidence files. |
| Worker route/custom domain | Provides a professional download domain such as `download.sourcedeck.com`. |
| Worker secrets | Stores webhook, token, and product mapping secrets outside git. |

## Required Secrets and Variables

| Name | Type | Example |
|---|---|---|
| `LEMONSQUEEZY_WEBHOOK_SECRET` | Secret | From Lemon Squeezy webhook settings. |
| `DOWNLOAD_SIGNING_SECRET` | Secret | Long random value generated by password manager. |
| `SOURCEDECK_PRODUCT_ID` | Variable | Lemon Squeezy product ID. |
| `SOURCEDECK_VARIANT_IDS` | Variable | Comma-separated allowed variant IDs. |
| `SOURCEDECK_CURRENT_MAC_DMG_KEY` | Variable | `releases/mac/SourceDeck-1.1.0.dmg`. |
| `SOURCEDECK_CURRENT_MAC_SHA256` | Variable | SHA-256 from release evidence. |
| `SOURCEDECK_RELEASES` | R2 binding | Private artifact bucket binding. |

## Endpoints

| Endpoint | Method | Purpose |
|---|---|---|
| `/health` | `GET` | Confirms the Worker is reachable. |
| `/webhooks/lemonsqueezy` | `POST` | Receives and verifies Lemon Squeezy events. |
| `/download/mac?token=...` | `GET` | Serves the current signed/notarized `.dmg` if the token is valid. |

## Deployment Notes

The Worker template intentionally avoids committing account IDs, secrets, or production artifact paths. Create the Cloudflare project, bind R2, and set variables through Cloudflare dashboard, Wrangler, or the configured Cloudflare connector once account authorization is available.

## Acceptance Test

The release operator must verify that an invalid webhook signature is rejected, a valid Lemon Squeezy test event returns a short-lived download URL, the download token expires, and the downloaded artifact hash matches `SOURCEDECK_CURRENT_MAC_SHA256`.
